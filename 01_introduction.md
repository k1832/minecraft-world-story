# マインクラフトの世界観・ストーリー考察：はじめに

マインクラフト（Minecraft）には、公式から提示される明確なストーリーやシナリオモードは存在しません。しかし、世界中を探索すると、廃坑、遺跡、海底神殿、朽ちたネザー要塞など、**「かつてそこに誰かがいて、文明を築いていた痕跡」**が数多く見つかります。

公式はあえて物語の全貌を語らず、「環境ストーリーテリング（背景やオブジェクトの配置から物語を想像させる手法）」を用いています。そのため、世界中のプレイヤーがゲーム内の建造物、Mobの生態、アイテムの由来などを手がかりに、独自の歴史や世界観を考察しています。

このドキュメント群では、世界中のコミュニティ（海外の「Game Theory」などの有名考察や、日本のコミュニティでの議論など）で支持されている代表的なマインクラフトのストーリー・世界観考察をテーマ別にまとめています。
